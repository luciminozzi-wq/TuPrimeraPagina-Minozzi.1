# Laboratorio Clínico Minozzi - TP Django

### Orden de prueba:
1. **Inicio:** Pantalla de presentación del laboratorio.
2. **Pacientes:** Ir a la sección "Pacientes", usar el botón verde para registrar uno nuevo.
3. **Estudios:** Ver el catálogo de estudios disponibles.
4. **Cargar Resultados:** Subir un PDF asociándolo a un paciente y un estudio.
5. **Buscador:** En la sección Pacientes, buscar por DNI para filtrar la tabla.
